# **App Name**: ClipBoard+

## Core Features:

- Detect Copied Text: Automatically detect and register copied text snippets.
- Snippet Saving: Save text snippets automatically to localStorage.
- Tagging and Categorization: Allow users to tag and categorize snippets for easy management.
- One-Click Copy: Enable copying snippets back to the clipboard with a single click.
- Local Storage History: Maintain a history of saved snippets within localStorage.

## Style Guidelines:

- Primary color: Dark gray (#333333) for a developer-themed dark UI.
- Background color: Very dark gray (#1A1A1A) to provide a clear separation between the content cards and the surrounding space, and establish the requested dark scheme.
- Accent color: Teal (#46D1C4) to complement the dark interface. The color suggests clean sophistication without cliche.
- Body and headline font: 'Source Code Pro' monospaced font for a developer-centric feel.
- Use clean card layouts to display snippets.
- Implement smooth hover effects for improved user interaction.
